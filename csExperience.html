  • Totem Busters

    • Start Data: March 14th, 2025
    • Last Worked On: June 26th, 2025
    • Progress: 629 Commits
    • Skills Utilized: Unity, C#, JSON Save/Load, SFX

    • History: This game began as my project for Clemson Forge 2025. This project is the first time I invested a significant amount of time to game development.
    • Description: As of May 26th, 2025, the game is a 3D turn-based fantasy role playing game developed in Unity. The game includes a start menu, options menu with SFX and save/load controls, a functional JSON save/load system, shops to purchase items, an inventory system, a character building system, customizable levels, a card game like battle system based on turns and rounds, multiple allies and combatants, and victory/loss states.     • Turbo Toads

      • Last Worked On: June 26th, 2025
      • Progress: 116 Commits

      • History: I chose to develop a second game in Summer 2025 to expand the skill range. I also wanted to revisit Godot as I was fairly inexperienced last time I used it.
